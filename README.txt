Реализация lock free sorted linked list. Первый и последний элемент в списке есть изначально.
Их значения задают минимальное и максимальное значение элементов , которые могут быть в списке.
Структура nodeType класса LinkedListIterator это структура узлов списка. Ссылка на след. элемент списка - link, значение(ключ) - info.

Классы и их краткое описание:

LinkedListIterator - класс итератор для списка, здесь мы объявляем основные операции, которые можно делать со списком и структуру узлов
связного списка

LinkedListType - абстрактный класс для списка как сортированного так и не сортированного, так как сортированный список делался
из не сортированного. Так же в классе описываются конструктор и деструктор списка, а отсортированный список просто наследует этот класс.

OrderedLinkedList - классс для отсортированного Lock Free списка. В нем объявляем и описываем методы, для неблокирующих операций
вставки, поиска и удаления элемента из списка.
